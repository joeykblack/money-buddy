# Ryan Gosling Theme Images

Add your Ryan Gosling images here with the following naming convention:

- 1.jpg
- 2.jpg
- 3.jpg
- ... up to 12.jpg

These images will be randomly selected when using the Ryan Gosling theme.

**Image Requirements:**
- Format: JPG, PNG, WebP
- Recommended size: 560x420 pixels (aspect ratio 4:3)
- The app will scale images to fit, so any reasonable resolution works
